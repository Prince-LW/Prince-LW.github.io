Cyclistic Project

For my data analysis project, I worked with a data set provided by Cyclistic, a fictional company. The goal of the project was to solve a specific business problem by combining and analyzing data using SQL, Tableau, and R.
To begin, I cleaned and preprocessed the data to ensure it was suitable for analysis. Next, I developed research questions to guide my analysis and used SQL to query and join different tables and databases to create a single dataset.
Using Tableau and R, I then analyzed the data to draw insights and make data-driven decisions. Based on my findings, I developed a plan to increase the number of users for Cyclistic's service.
Overall, the project allowed me to demonstrate my proficiency in SQL, Tableau, and R and showcase my ability to use data to solve business problems.
